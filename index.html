About Student Management System

This website is a simple Student Management System that allows users to add student information and view all stored student records. It's built using HTML, CSS (Tailwind), and JavaScript, demonstrating basic CRUD operations and data management in a web application.

Java Files:

Student.java: This file defines the Student class, which handles individual student data and file operations. It includes methods for accepting student information, saving it to a file, and printing all student information from the file.

StudentMenu.java: This file contains the main method and manages the program's menu-driven interface. It creates a Student object and provides options to add a student, print all students, or exit the program.
